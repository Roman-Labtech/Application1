О нас

"Лабораторные Технологии" - производитель пробоотборников для газа ПГО-100, ПГО-400, ПГО-50, ПУ-50, ПУ-400. Мы производим различные типы пробоотборников для газа, для нефтепродуктов, для красок, для пищевых продуктов. Нами выпускаются аторговой марки АГАТ Микрошприцы М-10 (аналог МШ-10), М-1 (аналог МШ-1), М-50, Газохром (различного объема).

  • Емкости из нержавеющей стали
  • Вискозиметры ВЗ-246 и ВЗ-4
  • Запорная арматура и вентили
  • Бомбы Рейда
